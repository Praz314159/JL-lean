# `JL/Research/assets/` — figures for the Conjecture 1 soundness argument

Explanatory figures for the Case 3 (Berry–Esseen) anti-concentration core of the mod-`q`
soundness bound. Each `.png` is regenerated by the sibling `.py` (matplotlib; `q` normalised to `1`).
Constants match the proof: `esseenConst = 0.56`, bad-arc half-width `c ≤ q/30`, sub-witness window
`‖v‖₂ ∈ [q/11, q/10)`, giving per-row failure `p ≈ 0.69 < 1`.

- **`jl_circle3d.{png,py}`** — the distribution of `⟨r,w⟩ mod q` as a density curtain on the circle
  `ℤ/qℤ`. The bump is near-Gaussian (Berry–Esseen, spread `σ ≈ q/15`); it is wider than the bad arc
  `|x| ≤ c` (red), so at most `≈40%` of the mass (`p₀`) can be forced into it — and the antipodal
  wrap region is `≈7σ` away. This is *what `p₀` is*, geometrically.
- **`jl_valley.{png,py}`** — per-row failure `p = p₀ + 2δ + τ` as a function of sub-witness size.
  A valley: too-small `σ` ⇒ the bump fills the arc and is too non-Gaussian (`p₀, 2δ → 1`); too-big
  `σ` ⇒ it wraps (`τ → 1`). The window `[q/11, q/10)` parks `σ` in the sweet spot; the valley floor
  shows the spendable margin (`1 − p`), tradeable for fewer rows or stronger soundness.

Regenerate: `python3 jl_circle3d.py` / `python3 jl_valley.py` (needs `numpy`, `scipy`, `matplotlib`).
